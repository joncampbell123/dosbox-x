00084 // The higher this number, the more memory will be used, but the more samples can be processed in one run -
00085 // various parts of sample generation can be processed more efficiently in a single run.
00086 // A run's maximum length is that given to Synth::render(), so giving a value here higher than render() is ever
00087 // called with will give no gain (but simply waste the memory).
00088 // Note that this value does *not* in any way impose limitations on the length given to render(), and has no effect
00089 // on the generated audio.
00090 // This value must be >= 1.
00091 const unsigned int MAX_SAMPLES_PER_RUN = 4096;
00092 
00093 // This determines the amount of memory available for simulating delays.
00094 // If set too low, partials aborted to allow other partials to play will not end gracefully, but will terminate
00095 // abruptly and potentially cause a pop/crackle in the audio output.
00096 // This value must be >= 1.
00097 const unsigned int MAX_PRERENDER_SAMPLES = 1024;
